Numpy¶

  • Numpy is a library in python that is used for scientific computing
  • It allows us to use high-performance multidimensional array objects
  • In this tutorial we will only go over the very basics of numpy

Arrays¶

  • A numpy array is a grid of values of the same type

Let's take a look:

In [1]:
import numpy as np #np is the standard convention we use for numpy

ra = np.array([1,2,3]) #rank 1 array (1D array)

print(ra)

[1 2 3]

Array components¶

Let's evaluate some of the compenents that allow us to see our array

  • Type: The type returned will bu ndarray which means an n-dimensional array
  • Shape: The shape will show us a tuple of the dimensions on the array
In [2]:
print(type(ra)) #Shows us the type of array
print(ra.shape) #One dimensional so should only print one dimension

<class 'numpy.ndarray'>
(3,)

Functions to Create Arrays¶

We can create arrays quickly using different numpy functions

  • zeros: Prints out an array of all zeroes with specified dimensions
  • ones: Prints out array of ones with specified dimensions
  • full: Prints out array of specified value and of specified dimensions
  • eye: Prints identity matrix
  • random.random: Creates array filled with random values of specified dimension
In [3]:
zero_ra = np.zeros((3,2)) #Array of zeroes 3 dimensions 2 elements in each dimension
print("zero array prints out: \n", zero_ra)

ones_ra = np.ones((1,2)) #1 dimension 2 element in dimension
print("ones array prints out: \n", ones_ra)

full_ra = np.full((2,3), 5) #2 dimensions, 3 elements per dimension, filled with 5's
print("full array prints out: \n" , full_ra)

eye_ra = np.eye(2) #2x2 identity matrix
print("eye array prints out: \n", eye_ra)

rand_ra = np.random.random((2,2)) #2 dimensions, 2 elements in dimensions, random values
print("random array prnts out: \n", rand_ra)

Different Graph Types¶

We're not going too much into the customizability of the graphs but I will show you different types of graphs:

  • Bar Chart
  • Pie Charts
  • Histogram
  • Scatter Plots

Bar Chart¶

A bar chart is typically used when you want to compare metric values accross different subgroups in your data
To create a bar chart use the plt.bar function in order to create one

  • bar(): The bar function takes a couple of arguements bar(different categories, caterogy values, color='insert color')
  • barh(): this bar function displays bar chart horizontally and uses the same arguements
In [31]:
courses = ["CS 596", "Stats 350A", "Ling 571", "CS 530", "CS 370"] #Declare categories
course_average_grades = [85, 82, 92, 75, 68] #Category values

plt.bar(courses, course_average_grades, color='blue') #Provide arguements to create bar chart
plt.title("Average Course Grades") #create labels
plt.xlabel("Courses")
plt.ylabel("Average Grades")
plt.show() #show
In [39]:
plt.barh(courses, course_average_grades, color='blue') #create chart
plt.title("Average Course Grades") #Add labels
plt.xlabel("Average Grades")
plt.ylabel("Courses")
plt.show() #display

The Actual potential of matplotlib¶

Above we showed basic functions in order to plot graphs. However, you can customize graphs a lot more than that, below is a quick showcase of these advanced graphs

import pandas as pd
import numpy

In [25]:
df = pd.read_csv("https://github.com/selva86/datasets/raw/master/mtcars.csv")
x = df.loc[:, ['mpg']]
df['mpg_z'] = (x - x.mean())/x.std()
df['colors'] = ['red' if x < 0 else 'green' for x in df['mpg_z']]
df.sort_values('mpg_z', inplace=True)
df.reset_index(inplace=True)

# Draw plot
plt.figure(figsize=(14,14), dpi= 80)
plt.hlines(y=df.index, xmin=0, xmax=df.mpg_z)
for x, y, tex in zip(df.mpg_z, df.index, df.mpg_z):
    t = plt.text(x, y, round(tex, 2), horizontalalignment='right' if x < 0 else 'left', 
                 verticalalignment='center', fontdict={'color':'red' if x < 0 else 'green', 'size':14})

# Decorations    
plt.yticks(df.index, df.cars, fontsize=12)
plt.title('Diverging Text Bars of Car Mileage', fontdict={'size':20})
plt.grid(linestyle='--', alpha=0.5)
plt.xlim(-2.5, 2.5)
plt.show()
