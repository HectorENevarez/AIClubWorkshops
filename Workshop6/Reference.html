threshold()¶

Thresholding an image helps with segmentation. We change the pixels of an image to become binary, black or white. That way we can detect the objects in the image
Lets take a look at the functions paramters
cv2.threshold(image, threshold, maxVal, type_of_threshold)

  • Image: This argument is the grayscale image you wish to use
  • threshold: if pixel value is less than threshold it will be set to 0
  • maxVal: any value is greater than the threshold it will be set to this maxVal
  • type_of_threshold: set what kind of threshold type you want.

Since our image is simple we use simple thresholding techniques. The code we use for thresholding works well with our basic image but you'll need more sophisticated techniques for thresholding more complex images. The full threshold documentation can be found here.

A couple things to notice from our code below

  1. the threshold returns a tuple. That's why in our code we add a '[1]' at the end because we are only interested in the image which happens to be in the 1 index
  2. We use the 'THRESH_BINARY_INV' type for our thresholding method. This inverts blacks and whites
In [3]:

findCountours()¶

This function helps to find all the white pixels in the image so that we can highlight the objects
Let's take a look at the function parameters:
cv2.findContours(image, mode, method)

  • image: thresholded image
  • mode: Contour retrieval model. All modes listed here
  • Method: Contour approximation method. All methods listed here

Let's take a look at the methods and modes we used for out image

  • RETR_EXTERNAL mode: This mode retrieves extreme outer contours. We use this mode because we are detecting for the edges of a shape so this contour mode works perfectly for our application. You can find more information here
  • CHAIN_APPROX_SIMPLE method: This method takes the contour and only saves the corners of each shape. That way we don't have all the edges stored but rather just the corners of the shapes. you can find more information here
In [4]:
cnts = cv2.findContours(thresh.copy(), cv2.RETR_EXTERNAL, cv2.CHAIN_APPROX_SIMPLE)
